`v0.8.4` is an evidence/testing checkpoint for the v0.8.x Sidecar flow. It
prepares external receiver smoke inputs and records public-safe status without
adding a new CLI command, API, schema, provider smoke, runtime integration,
tag, release, or npm publish step.

- Generated ignored private `generic` and `openclaw` sidecar smoke bundles.
- Ran `sidecar-check` against both targets.
