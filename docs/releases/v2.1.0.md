## What This Line Is

BaseBrief v2.1 is Context Pack Check: a narrow validation layer for Context
Pack Lite directories under the existing Artifact Checker surface.

The check stays on:

```text
node scripts/basebrief.js check --input <context-pack-dir> --json
```

It verifies whether a detected Context Pack Lite directory is complete enough,
public-safe enough, and explicit enough for receiver review. It does not prove
that every repository fact in the pack is true. It only verifies that the pack
exposes the expected files, shared metadata, live-fact anchors, missing-input
semantics, risk boundaries, starter instructions, and conservative thickness
signals for inspection.
