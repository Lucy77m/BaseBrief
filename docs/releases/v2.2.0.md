## What This Line Is

BaseBrief v2.2 is a narrow resume surface for checked Context Pack Lite
directories:

```text
node scripts/basebrief.js resume --input <context-pack-dir> [--json]
```

The command reads an existing Context Pack Lite directory, reuses the existing
Context Pack Check behavior, and prints copyable new-window prompt text.

- Clean Context Pack inputs produce a copyable prompt.
- Warning-only Context Pack inputs still produce a prompt and carry warning
  findings as review notes.
- Errored Context Pack inputs stop before prompt output.
- Human output is prompt text.
- JSON output includes the prompt because the prompt is the primary output.
