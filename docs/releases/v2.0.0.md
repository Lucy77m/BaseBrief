Date: 2026-06-08

## What This Line Is

BaseBrief v2.0 is Context Pack Lite: a local, reviewable, rule-based, file-based
context pack for the next coding-agent window.

The line produces seven Markdown artifacts:

```text
MANIFEST.md
REPO_MAP.md
KEY_FILES.md
RECENT_DELTA.md
RISK_BOUNDARIES.md
RECEIVER_STATE.md
NEXT_WINDOW_STARTER.md
```

The goal is not to dump more repository content into an AI window. The goal is
to make the minimum useful continuation context explicit: live facts, key
files, recent delta, receiver state, risk boundaries, review status, source,
trust, staleness, missing inputs, and the next-window starter.
