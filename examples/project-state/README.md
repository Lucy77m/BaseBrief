# Project State Example

This directory shows the public-safe shape of `.basebrief/state.json` after a
reviewed receiver-ready handoff is accepted into local project state.

The example uses `basebrief-project-state-v1` and stores no private absolute
paths, API keys, provider output, or `.env` data.

The state file is local continuity evidence only. It is not Auto Flow, does not
create a receiver thread, and does not make provider requests.
