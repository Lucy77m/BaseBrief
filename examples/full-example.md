# Full Example

## Scenario

A medium-complexity product workspace just finished one phase and needs a stable baseline for the next window.

## Request

```text
Please use BaseBrief to prepare a full phase baseline.
I also need a short next-chat opener and an agent task handoff.
```

## Expected Mode

- `full`

## Expected Output Shape

- `BASEBRIEF.md`
- optionally `NEXT_CHAT_PROMPT.md`
- optionally `AGENT_TASK.md`
- optionally `RISK_NOTES.md`
